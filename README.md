# slot-machine Web app
Interactive Slot Machine web app built with HTML, CSS, and JavaScript featuring a modern UI and random spin logic.

---

##  Features

-  3x3 slot machine grid
-  Random spin logic using JavaScript
-  Win detection system
-  Modern and responsive UI
-  Smooth hover effects and styling
-  Mobile-friendly design

---

##  Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Media Queries)
- JavaScript (DOM Manipulation)

---

##  Project Structure

slot-machine/
│── index.html
│── style.css
│── script.js
